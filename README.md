# Portfolio Site

This is a portfolio project I created to showcase my projects with various frameworks and tools. I initially built as an online resume with lots of animations, but have since cleaned it up for accessibility. The app is responsive, with clean functionality, and structured for seo.

Check it out live here: https://mgahelrasoul.com/

---

## 💻 Technologies

- `Next.js`
- `React.js`
- `TypeScript`
- `HTML`
- `CSS`
- `TailwindCSS`
- `Motion`
- `ReactBits`
- `Three.js`
- `Emailjs`

---

## ⚙️ Features

- **Hero**: Timed animation on mount, with easy access information including links.
- **Navigation**: Responsive Mobile and Desktop nav, with shortcuts to sections and smooth scroll.
- **About & Projects**: Reactive cards describing relevant skills and projects.
- **Contact**: `Emailjs` powered form, with `react-hot-toast` feedback.
- **Motion**: Implemented parallax, timed, on scroll, and on hover animations.
